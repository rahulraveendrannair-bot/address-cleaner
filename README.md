# Address Cleaning Agent

## Deploy to Streamlit Cloud (free, 5 minutes)

1. Fork or upload these files to a GitHub repository
2. Go to https://share.streamlit.io
3. Sign in with GitHub
4. Click "New app" → select your repo → set `app.py` as the main file
5. Click Deploy

Your users get a permanent URL like:
`https://yourname-address-cleaner-app-abc123.streamlit.app`

## Files
- `app.py` — the main application
- `requirements.txt` — Python dependencies (installed automatically by Streamlit Cloud)
